# ALOG - Gestão Financeira

## Descrição
Sistema web que permite cadastrar, atualizar e remover investimentos, reservas e despesas, consolidando as informações em uma dashboard interativa. O objetivo é fornecer ao usuário uma visão clara e imediata do seu status financeiro, auxiliando no controle das finanças.

## Funcionalidades
- Cadastro, atualização e exclusão de investimentos, reservas e despesas.
- Dashboard que apresenta de forma objetiva o status financeiro do usuário.
- Relatórios simples de receitas, despesas e saldo geral.
- Interface amigável para facilitar a navegação e gerenciamento de dados.

## Tecnologia Utilizada
- Linguagens: PHP, HTML, CSS e JS
- Banco de Dados: MySQL

## Como Rodar o Projeto
1. Faça o download dos arquivos do repositório.
2. Importe o banco de dados MySQL incluído no projeto (alog.sql).
3. Configure o arquivo de conexão com o banco de dados (config.php ou equivalente).
4. Abra a aplicação em um servidor local (XAMPP, WAMP ou similar).
5. Acesse via navegador: http://localhost/ALOG/LOGIN/login.php
6. Faça login e conheça a plataforma usando E-mail (adm@alog.com) e Senha (alog).

## Objetivo do Projeto
Demonstrar habilidades em desenvolvimento web, modelagem de banco de dados, programação em PHP e HTML/CSS, e criação de dashboards interativas, servindo como portfólio profissional para vagas de Analista de Sistemas Júnior ou Desenvolvedor.
